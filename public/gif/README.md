# GIF Placement

Drop your animated GIF here and name it `animation.gif`.

The fundraiser display will automatically load it and control its playback speed:

| Donation amount | Playback speed |
|-----------------|---------------|
| $1              | ~1× (normal)  |
| $10             | ~4× faster    |
| $25             | ~6× faster    |
| $100            | ~8× faster    |
| $250+           | 10× faster    |

The GIF gradually slows back to normal speed over ~10 seconds after each donation.

## Tips for best results

- **Looping GIFs** work best (the player loops automatically)
- Any size works — the display scales to fit
- Keep file size under ~5 MB for fast loading
